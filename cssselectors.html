Learn CSS

CSS stands for Cascading Style Sheets, and is used to add style to web documents. CSS describes how HTML elements should be displayed on screen, paper, or in other media.

CSS Selectors

CSS selectors are used to select the HTML elements that you want to style. There are many different types of selectors, including:

  • Element selectors
    •      p {
            color: red;
        }

    All paragraphs should be red because of the element selector.

  • Class selectors
    • .example {
            font-weight: bold;
        }

    This text should be bold because of the class selector.

  • ID selectors
    • #example {
            text-decoration: underline;
        }

    This text should be underlined because of the ID selector.

  • Attribute selectors
    • a[target="_blank"] {
            color: blue;
        }

    This link should be blue because of the attribute selector.

    This link should be blue and open in a new tab because of the attribute selector.

  • Pseudo-classes
    • a:hover {
            font-size: 20px;
        }

    Hover over this link to see the font size increase because of the pseudo-class.

CSS Properties and Values

Once you have selected an HTML element, you can use CSS properties to style it. There are many different CSS properties, including:

  • color
    •         color: green;
        color: #00ff00;
        color: rgb(0, 255, 0)
  • background-color
    • background-color: blue;
    
        
    background-image: url("https://hips.hearstapps.com/colorful-ripe-tropical-fruits-top-royalty-free-image-995518546-1564092355.jpg");
    
        
    background: linear-gradient(to bottom, blue, green);
  • font-size
    • font-size: 16px;
    em is a relative unit of measurement that is based on the font size of the parent element. For example, if the font size of the parent element is 16 pixels, 1em is equivalent to 16 pixels. 
    font-size: 2.5em;
    pt stands for "points" and is a unit of measurement commonly used in print design and typography. It is equivalent to 1/72nd of an inch. 
    font-size: 12pt;
    percentages are relative to the parent element's font-size 
    font-size: 200%;
    rem is also a relative unit of measurement, but it is based on the root font size. This means that the font size specified in the html selector is used as the base value. 
    font-size: 1.3rem;
    font-size using keywords like small, medium, large, x-large, xx-large 
    font-size: xx-large;
    vw and vh are units of measurement that are based on the size of the viewport. vw represents 1% of the viewport's width, while vh represents 1% of the viewport's height. 
    font-size: 10vh;
  • text-align
    • { text-align: justify; }
{ text-align: left; text-align: justify; }

    Left align: Sets the text to align with the left side of its container. Center align: Sets the text to be centered horizontally within its container. Right align: Sets the text to align with the right side of its container. Justify align: Sets the text to be evenly distributed within its container, creating straight edges on both sides. Mixed align: Sets different alignment values for the first line and subsequent lines of text. These alignment properties can be applied to different HTML elements such as paragraphs, headings, lists, and more.

  • font-family

    • Some common font-family values:

    1. Arial
    2. Helvetica
    3. Times New Roman
    4. Times
    5. Courier New
    6. Courier
    7. Verdana
    8. Georgia
    9. Palatino
    10. Garamond
  • margin
  • padding
  • border
    1. Individual Properties
      • margin-top, margin-right, margin-bottom, margin-left
      • padding-top, padding-right, padding-bottom, padding-left
      • border-top, border-right, border-bottom, border-left
    2. Shorthand Properties
      • margin: 10px 20px; sets a 10px margin on the top and bottom of an element, and 20px on the left and right
      • border-top: 1px solid red; sets 1px solid red border on the top of an element
      • margin: sets margin for all sides
      • padding: sets padding for all sides
      • border: sets border for all sides

CSS Exercises

Practice your CSS skills with these exercises:

  1. Add a background color to the entire page
  2. Change the font size of all paragraphs to 18px
  3. Change the font color of all headings to blue
  4. Add a border to all images
  5. Change the background color of all unordered lists to yellow
  6. Add a background color to all table cells with the class "highlight"
  7. Change the text color of all links to red
  8. Add a margin to all paragraphs within a div with the class "content"
  9. Change the font weight of all elements with the class "bold" to bold
  10. Change the background color of the first child element of all unordered lists to lightblue
