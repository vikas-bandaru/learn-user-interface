Exercise 1

Create a webpage that has a header, two columns and a footer. The header and footer should have a background color of light blue, and the columns should have a background color of white. Use CSS grid to layout the page.

Exercise 2

Create a webpage with a header, a main section and a footer. The main section should have a background color of light gray. Inside the main section, create three divs with different background colors, and place them next to each other using CSS float. Inside each div, add an image and some text.

Exercise 3

Create a webpage with a header, a main section and a footer. Inside the main section, create a form with some input fields, a select menu and a submit button. Use CSS to style the form with a border, padding and margin, and use CSS to style the input fields, select menu and button with appropriate colors and fonts.
